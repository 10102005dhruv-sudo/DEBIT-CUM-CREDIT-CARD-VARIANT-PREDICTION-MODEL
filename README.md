# DEBIT-CUM-CREDIT-CARD-VARIANT-PREDICTION-MODEL

# Credit and Debit Card Variant Prediction using Machine Learning

This project focuses on predicting suitable credit and debit card variants for users based on their financial and behavioral attributes. A Random Forest classification model is used to perform multi-class prediction.

## Overview
The system takes user-related features such as age, income, credit score, debt, and spending patterns to recommend an appropriate card variant. The project also includes a basic web interface for user interaction.

## Key Features
- Data preprocessing and cleaning of large-scale dataset (~0.4 million entries)
- Handling categorical variables using One-Hot Encoding and Label Encoding
- Multi-class classification using Random Forest algorithm
- Basic deployment on a temporary web interface for user input and prediction

## How it Works
- Raw data is cleaned and preprocessed
- Categorical features are encoded into numerical format
- A Random Forest model is trained on the processed dataset
- User inputs are passed to the model to generate predicted card variant

## Challenges Faced
- Handling inconsistent and non-numeric data
- Managing class imbalance across multiple card categories
- Structuring large-scale dataset for efficient model training

## Learning Outcomes
- Importance of data preprocessing in machine learning workflows
- Understanding of ensemble models like Random Forest
- Handling multi-class classification problems
- Exposure to deploying machine learning models for user interaction

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Google Colab


The model was also deployed on a temporary web interface for basic user interaction.
